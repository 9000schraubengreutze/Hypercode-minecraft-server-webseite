So spielst du mit

In drei Schritten auf dem Server.

Kopiere die Adresse, füge sie in Minecraft ein und los geht's. Für Java und Bedrock.

Java Edition PC · Mac · Linux
  1. 1
    Minecraft Java starten

    Öffne Minecraft Java Edition. Der Server läuft auf der Version 26.1.2, trete aber mit einer belieben bei.

  2. 2
    Server hinzufügen

    Gehe zu Mehrspieler → Server hinzufügen und gib die Adresse ein.

  3. 3
    Verbinden & spielen

    Doppelklick auf den Eintrag und du bist drin.

hypercodestudios.duckdns.org
Bedrock Edition Konsole · Handy · Windows
  1. 1
    Bedrock öffnen

    Starte Minecraft (Bedrock) auf deinem Gerät. Der Server läuft auf Version 26.1.2.

  2. 2
    Serverliste öffnen

    Gehe zu Spielen → Server und scrolle nach unten zu „Server hinzufügen“.

  3. 3
    Adresse eingeben

    Servername frei wählbar, Adresse wie unten — Standardport lassen.

hypercodestudios.duckdns.org

Tipp: Der Server läuft auf Version 26.1.2 — du kannst aber mit jeder neueren oder etwas älteren Minecraft-Version joinen. Speichere den Server einfach in deiner Liste.
